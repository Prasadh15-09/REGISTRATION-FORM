Attendance Calculator

How many more classes to attend to reach target percentage.

Enter values and click Compute to see how many more classes you need to attend.
Example with initial values:
Attended = 126, Held = 169, Target = 75% → Need to attend 3 more consecutive classes to reach exactly 75%.
Calculation: (126 + x)/(169 + x) ≥ 0.75 → x = ceil((0.75×169 −126)/0.25) = ceil(0.75/0.25)=3.
If you miss any of the next x classes, the required number will increase; recalc after each class.
